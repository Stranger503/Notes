# Homo-iconic

	* Homo-iconic means __Code as Data__
	* Code and Data are not differentiated which means Code is Data.
	* Ex: (add 2 3)
	* Just like above string, which is both a list and also a function call. The "Homo" prefix stands for this characteristic.
	* In other words we can say Homo-iconic means same representation inside and outside. This was derived from the word __homo__ means __same__.
